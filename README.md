# Pokédex

Buscador de Pokémon que consume la PokeAPI.

## Cómo usarlo
1. Abre el sitio desplegado.
2. Escribe el nombre de un Pokémon y presiona **Buscar** para agregarlo.
3. Usa el botón **Cargar más** para ver más Pokémon en la lista inicial.
4. Haz clic en **⚡ Capturar** para agregar un Pokémon buscado a tu colección.

## Tecnologías
- JavaScript (`fetch`, `async/await`, `Promise.all`)
- Tailwind CSS
- [PokeAPI](https://pokeapi.co/)
